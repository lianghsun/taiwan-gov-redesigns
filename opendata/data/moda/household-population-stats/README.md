# 戶役政人口結構統計

> ⚠️ **本資料完全為虛構示意**，數字並非真實之戶役政統計，僅作為示意 demo。

## 主權資料如何處理人口資料

戶役政是台灣最敏感、最完整的個人資料源之一。要讓研究者使用必須：

1. **聚合至安全粒度**：縣市 × 年度 × 年齡層
2. **k-anonymity 抑制**：當某格人數小於 k=5 時，以 `null` 取代避免再識別
3. **不揭露姓氏 / 戶長關聯**

## 限制

- 不可推估個人
- 不適用於村里級別研究（需另行申請）
- 「外籍配偶」「移工」等子群未細分
