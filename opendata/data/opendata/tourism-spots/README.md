# 全國觀光遊憩據點

> **示範資料**：本 demo 收錄 25 筆精選據點，正式資料集約 3,000+ 筆。

交通部觀光署的這份資料原本以「交通部觀光局推薦景點 OOS XML」格式提供，欄位中英混雜。本平台正規化為 JSONL，並補上 `category` 與 `ticketTwd` 兩欄方便篩選。

## 欄位

完整 schema 見右側「Schema」頁籤。重點：

- `category`：自然 / 文化 / 古蹟 / 主題園區 / 步道 / 夜市 / 海岸 / 溫泉
- `ticketTwd`：以新台幣計，免費景點為 `0`
- `lat` / `lng`：WGS84

## 應用情境

- 旅遊規劃 App
- 「免費景點地圖」社群分享
- 無障礙友善行程規劃
