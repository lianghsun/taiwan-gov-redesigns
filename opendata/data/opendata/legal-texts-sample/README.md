# 中央法規條文全文（精選）

> **示範資料**：本 demo 收錄與資訊公開、隱私、檔案管理相關的 5 部法律（含 1 部草案）的條文摘要。實務上的「全國法規資料庫」收錄超過 4,000 部法規。

## 為什麼以「條文 .txt」形式提供

`law.moj.gov.tw` 的條文以 HTML 與 PDF 為主，少部分提供 XML/JSON。但對 LLM、檢索系統、學術研究而言，最理想的形式是：

1. 一部法律 = 一個檔案
2. 純文字、UTF-8、條號統一格式（`第 X 條`）
3. 不夾雜 HTML 標籤、不含分頁錨點

本 demo 直接以這種「乾淨純文字」呈現。

## 檔案命名

`NN-法律名稱.txt`，數字為展示順序。每個檔案內含：

```
法律名稱
（修正日期）

第 1 條
條文內容…

第 2 條
條文內容…
```

## 與結構化資料的差別

這份資料沒有 schema、沒有欄位、沒有表格化預覽。它的「結構」是人類語言層次的（章節／條／項／款）。如果要做欄位式查詢，得靠下游程式（LLM 或自訂 parser）解析。
