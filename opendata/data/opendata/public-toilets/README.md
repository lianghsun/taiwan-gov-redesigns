# 全國公廁服務據點

> **示範資料**：本 demo 收錄 60 筆來自 6 個縣市的精選公廁，正式資料集約 4 萬筆。

## 為什麼需要這份資料

旅遊、外送、無障礙導覽、嬰幼兒外出，經常需要快速找到「最近、評等不差、有無障礙設施」的公廁。原 data.gov.tw 提供的格式為 CSV，但編碼不一、欄位混雜全形半形空白，本平台已正規化為 UTF-8 並統一座標系統為 WGS84。

## 欄位

| 欄位 | 型別 | 範例 |
| --- | --- | --- |
| `id` | string | `TPE-0001` |
| `name` | string | `捷運中山站 R7 出口` |
| `city` | string | `臺北市` |
| `lat`, `lng` | number | `25.0526, 121.5202` |
| `grade` | string | `特優` |
| `accessible` | boolean | `true` |
| `diaperStation` | boolean | `false` |

## 已知資料品質議題

- 偏鄉之臨時廁所未必納入
- `grade` 欄位由各縣市政府每年抽查，更新頻率不一
