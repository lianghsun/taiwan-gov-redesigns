# 中華民國 115 年（2026）政府行政機關辦公日曆表

> **示範資料**：本 demo 僅收錄 2026 年精選 30 天以呈現結構，正式資料應為全年 365 筆。

行政院人事行政總處每年公告該年度政府機關辦公日曆，是台灣許多排程、薪資、班表系統的源頭資料。原檔案在 data.gov.tw 是 CSV 格式，本平台提供乾淨的 **JSON** 形式，所有日期皆為 ISO 8601 (`YYYY-MM-DD`)，星期欄位採 ISO 8601 一致表示。

## 欄位說明

| 欄位 | 型別 | 說明 |
| --- | --- | --- |
| `date` | string (date) | 日期 |
| `weekday` | string | 星期一~星期日 |
| `isHoliday` | boolean | true=放假；false=辦公 |
| `description` | string | 備註說明 |

## 應用情境

- 排程系統判斷「下個工作日」
- 行事曆 / 班表 / 薪資結算
- 連假預測與旅遊熱度分析

## 與原版差異

- 原 data.gov.tw 提供 Big5 編碼的 CSV，本站轉為 UTF-8 並以 JSON 提供
- 補上 `isHoliday` 布林欄位，原檔案以「2」「0」混合字串表示
